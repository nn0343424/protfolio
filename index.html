Projects

BloodBank Management

BloodBank Management is a web application built using Django that streamlines the process of blood donation and management. The project features a robust search functionality, allowing users to quickly find available blood donors based on blood group and location. Individuals can easily register as donors, making their profiles accessible to those in need. The application also includes essential features such as donor registration, donor listing, and basic management tools for administrators. This project aims to simplify the connection between donors and recipients, making the process efficient and user-friendly.

Task Tracker App

Task Tracker App is a productivity tool built with React and Firebase. It allows users to create, edit, and delete tasks, set deadlines, and organize their workflow efficiently. Real-time updates and notifications help users stay on top of their daily goals. The app features a clean UI, drag-and-drop task management, and user authentication for secure access.
